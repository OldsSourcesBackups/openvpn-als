Change which keys function as the right mouse button

Show All

If you have a single-button mouse connected to the Mac, you can create your own combination of keys to function as the right mouse button. When you hold the keys down and click the mouse button, the Windows-based computer recognizes the action as a click of the right mouse button (right-click).

Note By default, Remote Desktop Connection uses CONTROL+SHIFT+click as the right-click function.

  1. On the RDC menu, click Preferences, and then click Keyboard.

  2. Next to Secondary Mouse Button, double-click the existing keyboard shortcut in the Mac column, and then press one or more modifier keys and a single-click, such as ⌘+SHIFT+click.
