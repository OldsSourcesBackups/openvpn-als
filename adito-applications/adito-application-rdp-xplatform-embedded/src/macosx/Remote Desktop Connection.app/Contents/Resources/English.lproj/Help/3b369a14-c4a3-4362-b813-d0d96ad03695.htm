Add your Windows user name and password to the Mac OS keychain

Show All

You can connect to a Windows-based computer more quickly by saving your login information to the Mac OS keychain. Information that is added to the keychain is saved in the connection fileA file that contains all of the information for a remote connection. For example, you can set the screen size for the session window, or share Mac drives with the Windows-based computer by saving these preferences to a connection file. If you connect to multiple Windows-based computers, you can save a different connection file for each computer., so that you don't have to enter it each time that you connect.

Note Passwords are saved to the Mac OS keychain only, not to the connection file.

  1. On the RDC menu, click Preferences.

  2. Click Login.

  3. Type the user name, password, and domain for the Windows-based computer, and then select the Add user information to your keychain check box.

    Tip You can also save partial user information to your keychain. For example, you can save the user name and domain so that you can log in by typing only the password.
