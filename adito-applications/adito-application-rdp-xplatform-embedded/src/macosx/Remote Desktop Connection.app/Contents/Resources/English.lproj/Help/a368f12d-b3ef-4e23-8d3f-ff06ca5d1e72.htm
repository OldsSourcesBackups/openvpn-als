I can't see the Mac menu bar

Show All

Cause: The Remote Desktop session is running in full-screen mode.

Solution: Point to the edge of the screen where the menu bar is usually located.

The menu bar is hidden in full-screen mode to maximize the available screen space.

Cause: Display preferences for full-screen mode are set not to show the menu bar and the Dock.

Solution: Clear the full-screen option in display preferences.

  1. On the RDC menu, click Preferences.

  2. Click Display, and then clear the Do not show the Mac menu bar and the Dock in full-screen mode check box.
