Travail en mode plein écran

Tout afficher

Lorsque vous vous connectez pour la première fois à un ordinateur Windows, la session à distance s'affiche sous forme de fenêtre sur votre bureau Mac. Vous pouvez passer du mode fenêtre au mode plein écran. Cela a pour effet d'agrandir l'écran pour n'afficher que le bureau de l'ordinateur Windows.

Effectuez l'une des opérations suivantes :

Passage en mode plein écran
Important

Après que vous êtes passé en mode plein écran, la barre de menus et le Dock Mac sont masqués. Pour afficher la barre de menus et le Dock, pointez sur le bord de l'écran où ils se trouvent habituellement. Pour revenir en mode Fenêtre ultérieurement, pointez sur le bord supérieur de l'écran jusqu'à ce que la barre de menus de Connexion Bureau à Distance s'affiche. Dans la barre de menus, cliquez sur Affichage, puis désactivez la case à cocher Plein écran.

  • Dans le menu Affichage, cliquez sur Plein écran.

    Conseil Lorsque vous pointez sur le Dock en mode plein écran, le Dock recouvre la barre des tâches de Windows. Pour empêcher que le Dock (et la barre de menus) s'affichent en mode plein écran et masquent la barre des tâches, modifiez vos préférences d'affichage Connexion Bureau à Distance.

Utilisation des raccourcis clavier en mode plein écran
