Impossible de copier et coller d'un ordinateur à l'autre

Tout afficher

Cause : Certains formats de presse-papiers ne peuvent pas être collés.

Solution : Copiez le texte dans un programme intermédiaire offrant un format de presse-papiers compatible.
Copie à partir d'un ordinateur Windows et collage sur un ordinateur Mac

  1. Sur l'ordinateur Windows, copiez le texte souhaité.

  2. Sur l'ordinateur Mac, ouvrez TextEdit et collez le texte.

  3. Copiez le texte dans TextEdit, puis collez-le dans le fichier de votre choix.

Copie à partir de l'ordinateur Mac et collage sur un ordinateur Windows

  1. Sur l'ordinateur Mac, copiez le texte de votre choix.

  2. Sur l'ordinateur Windows, ouvrez Bloc-Notes ou Wordpad, puis collez le texte.

  3. Copiez le texte dans Bloc-Notes ou Wordpad, puis collez-le dans le fichier de votre choix.

Cause : Le fichier est inaccessible sur l'ordinateur Windows.

Solution : Avant de démarrer la session Bureau à Distance, rendez les disques Mac disponibles pour l'ordinateur Windows.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Lecteurs, puis, dans le menu contextuel, sélectionnez l'option de votre choix.

    Votre disque ou dossier étant désormais disponible, vous pouvez accéder à des fichiers figurant sur votre ordinateur Mac à partir de votre ordinateur Windows.

    Si vous apportez cette modification après vous être connecté, le disque ou le dossier que vous sélectionnez devient disponible lors de votre connexion suivante.

  3. Dans le menu Démarrer, cliquez sur Ordinateur ou Mon ordinateur.

    Le disque ou dossier Mac que vous avez rendu disponible est répertorié avec tous les autres disques et dossiers de votre ordinateur Windows.

    Remarque Si vous ne pouvez pas afficher le menu Démarrer, dans le menu Afficher, cliquez sur Plein écran.Pour revenir en mode Fenêtre ultérieurement, pointez sur le bord supérieur de l'écran jusqu'à ce que la barre de menus de Connexion Bureau à Distance s'affiche. Dans la barre de menus, cliquez sur Affichage, puis désactivez la case à cocher Plein écran.

  4. Ouvrez le disque ou de dossier à partir duquel vous voulez copier.

  5. Dans le menu Démarrer, cliquez sur Ordinateur ou Mon ordinateur.

  6. Ouvrez le disque ou de dossier vers lequel vous voulez copier.

    Deux fenêtres devraient à présent être ouvertes : l'une pour l'emplacement source de la copie et l'autre pour l'emplacement cible de la copie.

  7. Faites glisser le document à copier vers l'emplacement de votre choix.

Cause : La stratégie de groupe de services Terminal ne permet pas un partage de lecteur.

Solution : Contactez l'administrateur réseau concernant votre stratégie de groupe de services Terminal.

La stratégie de groupe remplace toute préférence définie dans Connexion Bureau à Distance. Si la stratégie de groupe de services Terminal interdit le partage de lecteur, vous ne pouvez pas partager des fichiers entre des ordinateurs.

Cause : La stratégie de groupe de services Terminal ne permet pas un partage de presse-papiers.

Solution : Contactez l'administrateur réseau concernant votre stratégie de groupe de services Terminal.

La stratégie de groupe remplace toute préférence définie dans Connexion Bureau à Distance. Si la stratégie de groupe de services Terminal interdit le partage de presse-papiers, vous ne pouvez pas copier et coller des données entre des ordinateurs.
