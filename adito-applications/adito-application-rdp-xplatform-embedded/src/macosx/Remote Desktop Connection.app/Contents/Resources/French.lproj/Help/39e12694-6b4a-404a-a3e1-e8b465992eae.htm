Fermeture d'une connexion

Tout afficher

Vous pouvez fermer une session à distance soit en vous déconnectant, soit en fermant la session sur l'ordinateur Windows. La déconnexion interrompt la connexion mais laisse les programmes ou fichiers Windows que vous utilisez ouverts et actifs, afin que vous puissiez y revenir ultérieurement. La fermeture de session met fin à la session ; toute application en cours d'exécution dans la session est fermée et toute modification non enregistrée à des fichiers ouverts est perdue.

Fermeture d'une connexion sur un ordinateur exécutant Windows Vista

Pour vous déconnecter tout en permettant aux applications de continuer à s'exécuter sur l'ordinateur Windows, utilisez la commande Déconnecter. Pour vous déconnecter de Windows en fermant toutes les applications, utilisez la commande Fermer la session.

Déconnexion

  • Cliquez sur Démarrer, puis sur Déconnecter .

Fermeture de session

  • Cliquez sur Démarrer, sur la flèche à côté de Verrouiller , puis sur Fermer la session.

Fermeture d'une connexion sur un ordinateur exécutant Windows XP

Pour vous déconnecter tout en permettant aux applications de continuer à s'exécuter sur l'ordinateur Windows, utilisez la commande Déconnecter. Pour vous déconnecter de Windows en fermant toutes les applications, utilisez la commande Fermer la session.

Déconnexion

  • Cliquez sur Démarrer, sur Déconnecter  , puis de nouveau sur Déconnecter.

Fermeture de session

  • Cliquez sur Démarrer, sur Fermer la session, puis de nouveau sur Fermer la session.

Fermeture d'une connexion sur un ordinateur exécutant Windows Server 2003

Pour vous déconnecter tout en permettant aux applications de continuer à s'exécuter sur l'ordinateur Windows, utilisez la commande Déconnecter. Pour vous déconnecter de Windows en fermant toutes les applications, utilisez la commande Fermer la session.

Déconnexion

  • Dans la fenêtre de session Bureau à Distance, cliquez sur , puis sur Déconnecter.

Fermeture de session

  • Cliquez sur Démarrer, sur Fermer la session, puis de nouveau sur Fermer la session.
