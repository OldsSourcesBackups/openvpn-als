Création et modification de fichiers de connexion

Tout afficher

Connexion Bureau à Distance enregistre vos préférences de connexion dans un fichier de connexion par défaut. Pour gagner du temps lorsque vous vous connectez, vous pouvez enregistrer un nom d'ordinateur et des préférences de connexion supplémentaires pour chaque ordinateur Windows auquel vous vous connectez. Par exemple, vous pouvez choisir de partager tous les disques Mac lorsque vous vous connectez à un ordinateur Windows donné, et un seul dossier Mac lorsque vous vous connectez à un autre ordinateur Windows. Vous pouvez également enregistrer des fichiers de connexion différents pour le même ordinateur Windows.

Effectuez l'une des opérations suivantes :

Création d'un fichier de connexion

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

    Remarque Vous ne devez pas nécessairement taper des informations utilisateur dans le volet Connexion pour maintenir les préférences de paramétrage.

  2. Sélectionnez les préférences de connexion souhaitées pour cette connexion, telles que la taille d'écran, les informations de connexion automatique et les options de sécurité.

  3. Dans le menu Fichier, cliquez sur Enregistrer sous.

  4. Tapez un nom pour le fichier de connexion, puis cliquez sur Enregistrer.

    Lorsque vous établissez une connexion, les options que vous avez sélectionnées deviennent les préférences par défaut pour toutes les connexions Bureau à Distance.

Conseil Pour vous connecter rapidement à un ordinateur Windows pour lequel vous avez enregistré un fichier de connexion, dans le menu Fichier, cliquez sur Ouvrir une connexion enregistrée, puis cliquez sur le fichier de votre choix.

Modification d'un fichier de connexion

  1. Dans Connexion Bureau à Distance, dans le menu Fichier, cliquez sur Modifier une connexion.

  2. Sélectionnez le fichier de connexion souhaité, puis cliquez sur Ouvrir.

  3. Apportez les modifications souhaitées.

  4. Dans le menu Fichier, cliquez sur Enregistrer.

    Lorsque vous établissez une connexion, les options que vous avez sélectionnées deviennent les préférences par défaut pour toutes les connexions Bureau à Distance.

    Conseil Vous pouvez également modifier des préférences de connexion en cours de session. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences, puis apportez les modifications de votre choix. Les modifications prendront effet lors de votre prochaine connexion.
