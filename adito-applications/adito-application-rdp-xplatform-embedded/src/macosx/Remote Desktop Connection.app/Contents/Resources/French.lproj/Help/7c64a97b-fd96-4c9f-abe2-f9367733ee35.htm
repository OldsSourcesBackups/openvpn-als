Impossibilité d'accéder aux disques durs Mac

Tout afficher

Cause : Le disque dur Mac n'est pas partagé avec l'ordinateur Windows.

Solution : Sélectionnez le disque ou le dossier Mac à mettre à la disposition de l'ordinateur Windows.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Lecteurs, puis, dans le menu contextuel, sélectionnez l'option de votre choix.

    Votre disque ou dossier étant désormais disponible, vous pouvez accéder à des fichiers figurant sur votre ordinateur Mac à partir de votre ordinateur Windows.

  3. Sur l'ordinateur Windows, cliquez sur Démarrer, puis sur Ordinateur ou Poste de travail.

    Le disque ou dossier Mac que vous avez rendu disponible est répertorié avec tous les autres disques et dossiers de votre ordinateur Windows.

Cause : La stratégie de groupe de services Terminal ne permet pas un partage de disque.

Solution : Contactez l'administrateur réseau concernant votre stratégie de groupe de services Terminal.

La stratégie de groupe remplace toute préférence définie dans Connexion Bureau à Distance.Si la stratégie de groupe de services Terminal interdit le partage de disque, vous ne pouvez pas utiliser vos disques Mac sur l'ordinateur Windows.
