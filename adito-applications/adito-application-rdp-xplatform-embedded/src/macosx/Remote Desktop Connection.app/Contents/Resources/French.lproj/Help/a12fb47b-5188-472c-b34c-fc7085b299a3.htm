Connexion à un ordinateur Windows

Tout afficher

Avant de vous connecter, vous devez activer le paramètre de bureau à distance sur l'ordinateur auquel vous voulez vous connecter. Les procédures suivantes décrivent la procédure à suivre pour se connecter à un ordinateur Windows à partir de votre ordinateur Mac.

Étape 1 : autorisation des connexions à distance sur l'ordinateur Windows

Choisissez l'une des procédures suivantes, selon que l'ordinateur Windows auquel vous voulez vous connecter exécute Windows Vista ou Windows XP.
Autorisation des connexions à distance sur un ordinateur exécutant Windows Vista
Important

Pour modifier les paramètres de l'ordinateur distant, vous devez être administrateur sur l'ordinateur Windows.

  1. Sur l'ordinateur Windows, cliquez sur Démarrer , puis sur Panneau de configuration.

    Si vous êtes en Affichage classique, dans le volet de navigation, cliquez sur Page d’accueil du Panneau de configuration.

  2. Cliquez sur Système et maintenance, puis sur Système.

  3. Dans le volet de navigation, cliquez sur Paramètres d'utilisation à distance, puis, sous Bureau à Distance, cliquez sur l'option de votre choix.

    Si vous êtes invité à fournir un mot de passe administrateur ou une confirmation, tapez le mot de passe ou fournisseur la confirmation.

    Remarque L'authentification au niveau du réseau améliore la sécurité parce qu'elle requiert que vous entriez vos informations de connexion Windows sur l'ordinateur Mac avant d'initier la connexion à distance. Pour plus d'informations sur l'authentification au niveau du réseau, voir l'Aide sur Windows Vista.

  4. Cliquez sur Sélectionner des utilisateurs.

  5. Effectuez l'une des opérations suivantes :

    Pour

    Procédez comme suit

    Vérifier que vous avez accès

    Assurez-vous que votre compte d'utilisateur figure dans la zone. Si vous êtes administrateur sur l'ordinateur, vous y êtes automatiquement répertorié.

    Ajouter un compte utilisateur

    Cliquez sur Ajouter, puis, dans le champ, tapez le nom de compte d'utilisateur de votre choix.

Autorisation des connexions à distance sur un ordinateur exécutant Windows XP
Important

Pour modifier les paramètres de l'ordinateur distant, vous devez être administrateur sur l'ordinateur Windows.

  1. Sur l'ordinateur Windows, cliquez sur Démarrer, puis sur Panneau de configuration.

    Si vous êtes en Affichage classique, dans le volet de navigation, cliquez sur Basculer vers l'affichage des catégories.

  2. Cliquez sur Performances et maintenance, puis sur Système.

  3. Sous l'onglet Distant, activez la case à cocher Autoriser les utilisateurs à se connecter à distance à cet ordinateur.

  4. Cliquez sur Choisir des utilisateurs distants.

  5. Effectuez l'une des opérations suivantes :

    Pour

    Procédez comme suit

    Vérifier que vous avez accès

    Assurez-vous que votre compte d'utilisateur figure dans la zone. Si vous êtes administrateur sur l'ordinateur, vous y êtes automatiquement répertorié.

    Ajouter un compte utilisateur

    Cliquez sur Ajouter, puis, dans le champ, tapez le nom de compte d'utilisateur de votre choix.

Étape 2 : recherche du nom de l'ordinateur Windows

Lorsque vous ouvrez Connexion Bureau à Distance, vous devez entrer le nom de l'ordinateur Windows auquel vous voulez vous connecter. Choisissez l'une des procédures suivantes, selon que l'ordinateur Windows auquel vous voulez vous connecter exécute Windows Vista ou Windows XP.

Recherche du nom d'ordinateur dans Windows Vista

  1. Sur l'ordinateur Windows, cliquez sur Démarrer , puis sur Panneau de configuration.

    Si vous êtes en Affichage classique, dans le volet de navigation, cliquez sur Page d’accueil du Panneau de configuration.

  2. Cliquez sur Système et maintenance, puis sur Système.

  3. Sous Paramètres de nom d’ordinateur, de domaine et de groupe de travail, notez le nom de l'ordinateur ou son nom complet si l'ordinateur Windows est membres d'un domaine.

    Lorsque vous ouvrez Connexion Bureau à Distance sur l'ordinateur Mac, vous devez entrer ce nom.

Recherche du nom d'ordinateur dans Windows XP

  1. Sur l'ordinateur Windows, cliquez sur Démarrer, cliquez avec le bouton droit sur Poste de travail, puis cliquez sur Propriétés.

  2. Sous l'onglet Nom de l'ordinateur, notez le nom complet de l'ordinateur.

    Lorsque vous ouvrez Connexion Bureau à Distance sur l'ordinateur Mac, vous devez entrer ce nom.

Étape 3 : ouverture d'une nouvelle connexion

Vous pouvez ouvrir une connexion à un ou plusieurs ordinateurs Windows simultanément.

Important

L'ordinateur Windows doit être allumé et disposer d'une connexion réseau. Vous ne pouvez pas établir une connexion avec un ordinateur en état de veille ou de veille prolongée. Assurez-vous donc que les paramètres de veille et de veille prolongée de l'ordinateur Windows sont définis sur Jamais. Pour plus d'informations sur l'apport de ces modifications, voir l'Aide de Windows. Assurez-vous que l'ordinateur auquel vous voulez vous connecter exécute une version de Windows prenant en charge les connexions à distance. Pour obtenir une liste complète, voir Compatibilité entre les versions de Windows.

  1. Sur l'ordinateur Macintosh, ouvrez Connexion Bureau à Distance.

  2. Dans le champ Ordinateur, tapez le nom de l'ordinateur Windows auquel vous voulez vous connecter, puis cliquez sur Connecter.

  3. Tapez vos nom d'utilisateur, mot de passe et domaine pour l'ordinateur Windows si nécessaire.
