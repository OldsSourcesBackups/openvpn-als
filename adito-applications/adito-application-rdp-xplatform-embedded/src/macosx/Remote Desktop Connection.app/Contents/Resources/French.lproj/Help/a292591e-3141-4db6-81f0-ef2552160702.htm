Limitation d'une session à une seule application Windows

Tout afficher

Si vous travaillez fréquemment dans une application spécifique sur un serveur exécutant Windows Server, vous pouvez gagner du temps en démarrant la session Bureau à Distance avec cette seule application. Des administrateurs peuvent également limiter des ordinateurs client à une seule application lorsqu'ils sont connectés à un serveur Windows. Le bureau Windowsest invisible ou indisponible lorsque cette préférence est sélectionnée.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences, puis sur Applications.

  2. Activez la case à cocher.

  3. Dans le champ Chemin d'accès et nom de fichier de l'application, tapez le chemin d'accès et le nom de fichier de l'application que vous voulez démarrer.

    Vous pouvez également taper le chemin d'accès du répertoire de travailDossier dans lequel une application est installée et s'exécute. La plupart des applications installées sont conçues avec un répertoire de travail fixe qui ne doit pas être modifié. Divers utilitaires, applications plus anciennes et autres fichiers de scripts et de traitement par lots sont conçus pour s'exécuter dans un dossier différent de celui dans lequel se trouve l'application. pour l'application si nécessaire.

Conseil Pour fermer la connexion, quittez l'application ou déconnectez-vous de l'ordinateur Windows.
