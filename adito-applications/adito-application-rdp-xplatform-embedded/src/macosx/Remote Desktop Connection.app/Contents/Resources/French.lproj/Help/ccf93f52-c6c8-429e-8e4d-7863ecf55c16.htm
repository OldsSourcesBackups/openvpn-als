Options d'accès universel

Connexion Bureau à Distance utilise un nombre sélectionné de fonctionnalités d'accès universel de Mac OS X. Ces dernières facilitent l'utilisation de l'ordinateur si vous êtes malvoyant ou malentendant ou en cas de dextérité limitée.

Pour définir des options d'accès, dans le menu Apple, cliquez sur Préférences système, puis, sous Système, cliquez sur Accès universel. Le tableau suivant répertorie les fonctionnalités d'accès universel de Mac OS X que vous pouvez utiliser dans Connexion Bureau à Distance.

Les fonctionnalités de Mac OS X n'opèrent pas sur le bureau Windows une fois la session Bureau à Distance établie. Pour plus d'informations sur les fonctionnalités d'accessibilité disponibles dans Windows, voir la page Web de Accessibilité Microsoft à l'adresse http://www.microsoft.com/enable.
