Impossibilité d'imprimer sur une imprimante connectée à l'ordinateur Mac

Tout afficher

Cause : L'imprimante connectée à l'ordinateur Mac n'est pas disponible pour l'ordinateur Windows.

Solution : Définissez les préférences d'imprimante pour rendre l'imprimante disponible.

  1. Dans le menu Connexion Bureau à Distance, cliquez sur Préférences.

  2. Cliquez sur Imprimantes, puis activez la case à cocher Utiliser une imprimante connectée au Mac.

  3. Sélectionnez l'imprimante de votre choix.

    Conseil Si vous voulez sélectionner une imprimante au moment de l'impression, sélectionnez Toutes les imprimantes.

Cause : La stratégie de groupe de services Terminal ne permet pas un partage d'imprimante.

Solution : Contactez l'administrateur réseau concernant votre stratégie de groupe de services Terminal.

La stratégie de groupe remplace toute préférence définie dans Connexion Bureau à Distance. Si la stratégie de groupe empêche le partage d'imprimante, vous ne pouvez pas imprimer de documents à partir de votre ordinateur Windows sur une imprimante connectée à l'ordinateur Mac.

Cause : Le câble de connexion de l'imprimante est mal connecté ou l'imprimante connectée à l'ordinateur Mac est éteinte.

Solution : Contrôlez l'état de l'imprimante.

  • Fixez le câble de connexion à l'imprimante, puis mettez celle-ci sous tension.

Conseil Pour vérifier si l'imprimante est connectée à l'ordinateur Mac, dans le menu Apple, cliquez sur Préférences système, puis, sous Matériel, cliquez sur Impression et télécopie.
