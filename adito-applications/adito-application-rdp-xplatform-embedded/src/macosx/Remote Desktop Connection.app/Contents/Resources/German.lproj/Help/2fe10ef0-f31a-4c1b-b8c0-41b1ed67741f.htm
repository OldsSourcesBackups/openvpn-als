Ändern der Tastenbelegung für die rechte Maustaste

Alles anzeigen

Wenn Sie eine Maus mit einer Taste an den Macintosh angeschlossen haben, können Sie eine eigene Tastenkombination festlegen, die die Funkion der rechten Maustaste übernimmt. Wenn Sie diese Tasten gedrückt halten und mit der Maustaste klicken, wird diese Aktion vom Windows-basierten Computer als Klicken mit der rechten Maustaste (Rechtsklick) interpretiert.

Hinweis Die Remotedesktopverbindung verwendet standardmäßig CTRL+UMSCHALT+Mausklick anstelle der rechten Maustaste.

  1. Klicken Sie im Menü RDC auf Einstellungen und anschließend auf Tastatur.

  2. Doppelklicken Sie neben Sekundäre Maustaste auf die vorhandene Tastenkombination in der Macintosh-Spalte. Drücken Sie eine oder mehrere Sondertasten, und klicken Sie einmal mit der Maustaste, z. B. ⌘+UMSCHALT+Mausklick.
