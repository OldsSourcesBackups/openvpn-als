Erstellen und Ändern von Verbindungsdateien

Alles anzeigen

Ihre Verbindungseinstellungen werden von der Remotedesktopverbindung in einer Standardverbindungsdatei gespeichert. Wenn Sie auf schnelle Weise eine Verbindung herstellen möchten, können Sie weitere Namen Windows-basierter Computer zusammen mit den jeweiligen Verbindungseinstellungen speichern. Beispielsweise können Sie alle Macintosh-Datenträger freigeben, wenn Sie eine Verbindung mit einem Windows-basierten Computer herstellen und nur einen Macintosh-Ordner freigeben, wenn Sie eine Verbindung mit einem anderen Windows-basierten Computer herstellen. Sie können auch mehr als eine Verbindungsdatei für den gleichen Windows-basierten Computer speichern.

Führen Sie eine der folgenden Aktionen aus:

Erstellen einer Verbindungsdatei

  1. Klicken Sie im Menü RDC auf Einstellungen.

    Hinweis Sie müssen keine Benutzerinformationen im Anmeldefenster eingeben, um den Vorgang zum Festlegen der Einstellungen fortzusetzen.

  2. Wählen Sie die gewünschten Verbindungseinstellungen wie Bildschirmgröße, automatische Anmeldeinformationen und Sicherheitsoptionen aus.

  3. Klicken Sie im Menü Datei auf Speichern unter.

  4. Geben Sie einen Dateinamen für die Verbindungsdatei ein, und klicken Sie auf Speichern.

    Sobald Sie eine Verbindung herstellen, werden die ausgewählten Optionen als Standardeinstellung für alle Remotedesktopverbindungen übernommen.

Tipp Wenn Sie auf schnelle Weise auf einen Windows-basierten Computer zugreifen möchten, für den Sie eine Verbindungsdatei gespeichert haben, klicken Sie im Menü Datei auf Gespeicherte Verbindung öffnen, und klicken Sie dann auf die gewünschte Datei.

Ändern einer Verbindungsdatei

  1. Klicken Sie in der Remotedesktopverbindung im Menü Datei auf Verbindung bearbeiten.

  2. Wählen Sie die gewünschte Verbindungsdatei aus, und klicken Sie auf Öffnen.

  3. Nehmen Sie die gewünschten Änderungen vor.

  4. Klicken Sie im Menü Datei auf Speichern.

    Sobald Sie eine Verbindung herstellen, werden die ausgewählten Optionen als Standardeinstellung für alle Remotedesktopverbindungen übernommen.

    Tipp Sie können die Verbindungseinstellungen auch während einer Sitzung ändern. Klicken Sie dazu im Menü RDC auf Einstellungen, und nehmen Sie die gewünschten Änderungen vor. Die Änderungen werden wirksam, wenn Sie das nächste Mal eine Verbindung herstellen.
