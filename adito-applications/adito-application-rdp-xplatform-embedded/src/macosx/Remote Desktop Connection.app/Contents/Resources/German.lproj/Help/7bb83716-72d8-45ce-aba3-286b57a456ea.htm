Herstellen einer Verbindung mit einer Serverkonsolensitzung

Alles anzeigen
Wichtig

Wenn Sie sich bei einer Konsolensitzung auf einem Remoteserver mit Windows Server anmelden möchten, müssen Sie auf dem Windows-basierten Computer als Administrator angemeldet sein.

  1. Öffnen Sie die Remotedesktopverbindung.

  2. Geben Sie im Feld Computer den Namen oder die IP-Adresse des Windows-basierten Computers ein, mit dem Sie eine Verbindung herstellen möchten, und geben Sie anschließend /console (beispielsweise FabricamFileServer/console) ein.

  3. Klicken Sie auf Verbinden.

  4. Geben Sie Ihren Benutzernamen, Ihr Kennwort sowie ggf. die Domäne ein.

Hinweis Diese Art der Anmeldung kann nur auf einem Server mit Windows Server ausgeführt werden.
