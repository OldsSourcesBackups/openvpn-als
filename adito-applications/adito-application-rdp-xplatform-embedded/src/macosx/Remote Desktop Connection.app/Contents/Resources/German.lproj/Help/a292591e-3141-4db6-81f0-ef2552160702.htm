Begrenzen einer Sitzung auf eine Windows-basierte Anwendung

Alles anzeigen

Wenn Sie häufig eine bestimmte Anwendung auf einem Server mit Windows Server verwenden, können Sie Zeit sparen, indem Sie die Remotedesktopsitzung nur für diese Anwendung öffnen. Netzwerkadministratoren können auch Clientcomputer auf eine Anwendung begrenzen, wenn diese mit einem Server mit Windows verbunden sind. Der Windows-Desktop ist weder sichtbar noch verfügbar, wenn diese Einstellung ausgewählt wird.

  1. Klicken Sie im Menü RDC auf Einstellungen und anschließend auf Anwendungen.

  2. Aktivieren Sie das Kontrollkästchen.

  3. Geben Sie im Feld Anwendungspfad und Dateiname den Pfad und den Namen der Anwendungen, die gestartet werden soll.

    Bei Bedarf können Sie auch den Pfad des ArbeitsverzeichnisDer Ordner, in dem eine Anwendung installiert ist und ausgeführt wird. Meist verfügt eine Anwendung über ein festgelegtes Arbeitsverzeichnis, das nicht mehr geändert werden muss. Einige ältere Anwendungen, Dienstprogramme und bestimmte Skript- und Batchdateien sind so konfiguriert, dass sie in einem anderen Ordner als dem Anwendungsordner ausgeführt werden. für die Anwendung eingeben.

Tipp Zum Trennen der Verbindung können Sie die Anwendung beenden oder sich bei dem Windows-basierten Computer abmelden.
