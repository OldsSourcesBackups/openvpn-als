Optionen für Bedienungshilfen

Die Remotedesktopverbindung verwendet eine Reihe von Bedienungshilfen von Mac OS X. Bedienungshilfen erleichtern die Verwendung des Computers, wenn Sie Schwierigkeiten beim Sehen oder Hören haben oder über eine eingeschränkte Fingerfertigkeit verfügen.

Klicken Sie zum Festlegen von Optionen für Bedienungshilfen im Menü Apple auf Systemeinstellungen und anschließend unter System auf Bedienungshilfen. Die folgende Tabelle enthält die Bedienungshilfen von Mac OS X, die Sie für die Remotedesktopverbindung verwenden können.

Mac OS X-Features können nach dem Herstellen der Remotedesktopsitzung nicht auf dem Windows-Desktop verwendet werden. Informationen zu den Bedienungshilfen von Windows finden Sie auf der Microsoft-Eingabehilfen-Webseite unter http://www.microsoft.com/enable.

VoiceOver
