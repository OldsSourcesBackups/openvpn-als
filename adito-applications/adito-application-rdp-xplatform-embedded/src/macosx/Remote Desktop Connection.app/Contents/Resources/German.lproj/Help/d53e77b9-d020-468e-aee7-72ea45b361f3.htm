Authentifizierungsoptionen

Bei der Remotecomputerauthentifizieriung wird überprüft, dass Sie eine Verbindung mit dem richtigen Windows-basierten Computer herstellen. Es stehen drei verschiedene Authentifizierungsoptionen zur Verfügung.

  • Immer Verbinden, auch wenn Authentifizierung fehlschlägt

    Auch wenn die Identität des Windows-basierten Computers durch die Remotedesktopverbindung nicht überprüft werden kann, wird eine Verbindung hergestellt. Diese Authentifizierungsoption weist die niedrigste Sicherheitsstufe auf.

    Hinweis Die Identität von Windows-basierten Computern, auf denen Windows XP oder früher oder Windows Server 2003 Service Pack 1 (SP1) oder früher ausgeführt werden, kann nicht verifiziert werden. Wenn Sie wissen, dass auf dem Windows-basierten Computer eines dieser früheren Betriebssysteme ausgeführt wird, können Sie Authentifizierungsprobleme umgehen, indem Sie Immer Verbinden, auch wenn Authentifizierung fehlschlägt auswählen.

  • Warnung anzeigen, falls Authentifizierung fehlschlägt (Standardeinstellung)

    Wenn die Identität des Windows-basierten Computer durch die Remotedesktopverbindung nicht überprüft werden kann, wird eine Warnmeldung angezeigt, damit Sie entscheiden können, ob die Verbindung hergestellt werden soll. Diese Authentifizierungsoption bietet eine höhere Sicherheitsstufe im Vergleich zur Option Immer Verbinden, auch wenn Authentifizierung fehlschlägt.

  • Keine Verbindung, wenn Authentifizierung fehlschlägt

    Wenn die Identität des Windows-basierten Computers durch die Remotedesktopverbindung nicht überprüft werden kann, kann keine Verbindung hergestellt werden. Diese Authentifizierungsoption bietet verbesserte Sicherheitsfeatures, wenn Sie eine Verbindung mit einem Computer unter Windows Vista herstellen. Im Vergleich zu den beiden anderen Authentifizierungsoptionen ist hier die höchste Sicherheitsstufe geboten.

    Hinweis Wenn Sie diese Option auswählen, können Sie nur Verbindungen mit Computern herstellen, auf denen Windows Vista oder eine frühere Version von Windows ausgeführt wird.
