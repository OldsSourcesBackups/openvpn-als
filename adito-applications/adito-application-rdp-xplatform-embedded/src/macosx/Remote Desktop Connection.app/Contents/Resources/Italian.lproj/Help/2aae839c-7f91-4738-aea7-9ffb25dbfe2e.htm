Non è possibile eseguire le operazioni di copia e incolla tra computer

Mostra tutto

Causa: Non è possibile incollare alcuni formati degli Appunti.

Soluzione: Copiare il testo in un programma intermedio con un formato Appunti compatibile.
Operazioni di copia e incolla da un computer Windows a un Mac

  1. Sul computer Windows copiare il testo desiderato.

  2. Sul Mac, aprire TextEdit e incollare il testo.

  3. Copiare il testo da TextEdit e incollarlo nel file desiderato.

Operazioni di copia e incolla da Mac a un computer Windows

  1. Sul Mac copiare il testo desiderato.

  2. Sul computer Windows, aprire Blocco note o WordPad e incollare il testo.

  3. Copiare il testo da Blocco note o WordPad e incollarlo nel file desiderato.

Causa: Non è possibile accedere al file dal computer Windows.

Soluzione: Prima di avviare la sessione di desktop remoto, rendere disponibili i dischi Mac per il computer Windows.

  1. Scegliere Preferenze dal menu Connessione desktop remoto.

  2. Fare clic su Unità, quindi dal menu a comparsa, scegliere l'opzione desiderata.

    Il disco o la cartella è disponibile. Adesso è possibile accedere ai file presenti su Mac dal computer Windows.

    Se si esegue questa modifica dopo la connessione, il disco o la cartella selezionata risulterà disponibile alla connessione successiva.

  3. Dal menu Start scegliere Computer o Computer locale.

    Il disco o la cartella Mac resa disponibile è elencata con tutti gli altri dischi e le altre cartelle sul computer Windows.

    Nota Se il menu Start non è visualizzato, scegliere Schermo intero dal menu Visualizza.Per tornare alla modalità finestra in un secondo momento, puntare sul bordo superiore della schermata finché non viene visualizzata la barra dei menu Connessione desktop remoto. Sulla barra dei menu fare clic su Visualizza, quindi deselezionare la casella di controllo Schermo intero.

  4. Aprire il disco o la cartella da cui si desidera eseguire la copia.

  5. Dal menu Start scegliere Computer o Computer locale.

  6. Aprire il disco o la cartella in cui si desidera eseguire la copia.

    A questo punto le finestre aperte dovrebbero essere due: una per il percorso da cui si desidera eseguire la copia e una per il percorso in cui si desidera eseguire la copia.

  7. Trascinare il documento da copiare nel percorso desiderato.

Causa: Criteri di gruppo di Servizi terminal non consente la condivisione delle unità.

Soluzione: Rivolgersi all'amministratore di rete per Criteri di gruppo di Servizi terminal.

Criteri di gruppo ignora eventuali preferenze impostate in Connessione desktop remoto. Quando Criteri di gruppo di Servizi terminal impedisce la condivisione di unità, non è possibile condividere file tra computer.

Causa: Criteri di gruppo di Servizi terminal non consente la condivisione di Appunti.

Soluzione: Rivolgersi all'amministratore di rete per Criteri di gruppo di Servizi terminal.

Criteri di gruppo ignora eventuali preferenze impostate in Connessione desktop remoto. Quando Criteri di gruppo di Servizi terminal impedisce la condivisione di Appunti, non è possibile copiare e incollare dati tra computer.
