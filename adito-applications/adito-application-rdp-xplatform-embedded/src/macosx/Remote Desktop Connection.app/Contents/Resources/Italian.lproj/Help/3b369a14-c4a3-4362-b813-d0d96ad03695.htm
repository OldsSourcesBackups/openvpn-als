Aggiunta della password e del nome utente di Windows al portachiavi di Mac OS

Mostra tutto

È possibile connettersi a un computer Windows più rapidamente salvando i dati di accesso nel portachiavi di Mac OS. I dati aggiunti al portachiavi vengono salvati nel file di connessioneFile contenente tutte le informazioni per la connessione remota. Ad esempio, è possibile impostare le dimensioni dello schermo per la finestra della sessione o condividere unità Mac con il computer Windows salvando queste preferenze in un file di connessione. Se ci si connette a più computer Windows, è possibile salvare un file di connessione diverso per ciascun computer.. In questo modo non sarà necessario digitarli a ogni connessione.

Nota Le password vengono salvate solo nel portachiavi di Mac OS e non nel file di connessione.

  1. Scegliere Preferenze dal menu Connessione desktop remoto.

  2. Fare clic su Accesso.

  3. Digitare il nome utente, la password e il dominio del computer Windows, quindi selezionare la casella di controllo Aggiungi informazioni utente al portachiavi.

    Suggerimento È possibile salvare nel portachiavi anche solo alcune informazioni utente. Ad esempio, è possibile salvare il nome utente e il dominio in modo da accedere digitando esclusivamente la password.
