Creazione e modifica dei file di connessione

Mostra tutto

In Connessione desktop remoto le preferenze di connessione vengono salvate in un file di connessione predefinito. Per risparmiare tempo durante la connessione, è possibile salvare ulteriori nomi computer e preferenze di connessione per ciascun computer Windows a cui ci si connette. Ad esempio, è possibile scegliere di condividere tutti i dischi Mac quando ci si connette a un computer Windows e solo una singola cartella Mac quando ci si connette a un computer Windows diverso. È, inoltre, possibile salvare file di connessione diversi per lo stesso computer Windows.

Eseguire una delle seguenti operazioni:

Creazione di un file di connessione

  1. Scegliere Preferenze dal menu Connessione desktop remoto.

    Nota Per continuare l'impostazione delle preferenze, non è necessario digitare informazioni utente nel riquadro di accesso.

  2. Selezionare le preferenze di connessione desiderate, come la dimensione dello schermo, le informazioni di accesso automatico e le opzioni di protezione.

  3. Scegliere Salva con nome dal menu File.

  4. Digitare un nome per il file di connessione, quindi fare clic su Salva.

    Una volta stabilita la connessione, le opzioni selezionate diventano le impostazioni predefinite per tutte le connessioni Desktop remoto.

Suggerimento Per connettersi rapidamente a un computer Windows per il quale è stato salvato un file di connessione, scegliere Apri connessione salvata dal menu File, quindi fare clic sul file desiderato.

Modifica di un file di connessione

  1. In Connessione desktop remoto scegliere Modifica connessione dal menu File.

  2. Selezionare il file di connessione desiderato e fare clic su Apri.

  3. Apportare le modifiche desiderate.

  4. Scegliere Salva dal menu File.

    Una volta stabilita la connessione, le opzioni selezionate diventano le impostazioni predefinite per tutte le connessioni Desktop remoto.

    Suggerimento È, inoltre, possibile modificare le preferenze di connessione durante una sessione. Scegliere Preferenze dal menu Connessione desktop remoto, quindi apportare le modifiche desiderate. Le modifiche verranno applicate alla connessione successiva.
