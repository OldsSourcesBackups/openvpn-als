Non è possibile connettersi al computer Windows

Mostra tutto

Causa: Il computer Windows non è impostato per accettare connessioni remote.

Soluzione: Attivare l'impostazione di desktop remoto sul computer a cui si desidera connettersi.
Attivare l'impostazione desktop remoto in Windows Vista
Importante

Le impostazioni di desktop remoto possono essere modificate solo dall'amministratore del computer Windows.

  1. Dal computer Windows fare clic su Start , quindi su Pannello di controllo.

    In Visualizzazione classica, nel riquadro di spostamento, fare clic su Pagina iniziale Pannello di controllo.

  2. Fare clic su Sistema e manutenzione, quindi su Sistema.

  3. Nel riquadro di spostamento fare clic su Impostazioni di connessione remota, quindi in Desktop remoto fare clic sull'opzione desiderata.

    Se viene richiesta la password amministratore o una conferma, digitare la password o confermare.

    Nota Autenticazione a livello di rete dispone di funzioni di protezione migliorate in Windows Vista ed è più sicuro rispetto alle altre due opzioni di autenticazione.

  4. Fare clic su Seleziona utenti.

  5. Eseguire una o più delle operazioni seguenti:

    A

    Eseguire questa operazione

    Verifica dell'accesso

    Accertarsi che l'account utente sia elencato sotto la casella. L'amministratore del computer viene elencato automaticamente.

    Aggiunta di un altro account utente

    Fare clic su Aggiungi, quindi, nella casella, digitare il nome dell'account utente desiderato.

Attivazione dell'impostazione desktop remoto in Windows XP
Importante

Le impostazioni di desktop remoto possono essere modificate solo dall'amministratore del computer Windows.

  1. Dal computer Windows fare clic su Start, quindi su Pannello di controllo.

    In Visualizzazione classica, nel riquadro di spostamento, fare clic su Passa alla visualizzazione per categorie.

  2. Fare clic su Prestazioni e manutenzione, quindi su Sistema.

  3. Sulla scheda Connessione remota selezionare la casella di controllo Consenti agli utenti di connettersi in remoto al computer.

  4. Fare clic su Seleziona utenti remoti.

  5. Eseguire una o più delle operazioni seguenti:

    A

    Eseguire questa operazione

    Verifica dell'accesso

    Accertarsi che l'account utente sia elencato sotto la casella. L'amministratore del computer viene elencato automaticamente.

    Aggiunta di un altro account utente

    Fare clic su Aggiungi, quindi, nella casella, digitare il nome dell'account utente desiderato.

Soluzione: Accertarsi che Windows Firewall consenta eccezioni per desktop remoto.
Consentire eccezioni attraverso Windows Firewall su un computer che esegue Windows Vista
Importante

Le impostazioni firewall possono essere modificate solo dall'amministratore del computer Windows.

  1. Dal computer Windows fare clic su Start , quindi su Pannello di controllo.

    In Visualizzazione classica, nel riquadro di spostamento, fare clic su Pagina iniziale Pannello di controllo.

  2. Fare clic su Protezione, quindi in Windows Firewall, fare clic su Consenti programma con Windows Firewall.

    Se viene richiesta la password amministratore o una conferma, digitare la password o confermare.

  3. Nell'elenco Programma o porta verificare che la casella di controllo Desktop remoto sia selezionata.

Consentire eccezioni con Windows Firewall su un computer che esegue Windows XP
Importante

Le impostazioni firewall possono essere modificate solo dall'amministratore del computer Windows.

  1. Dal computer Windows fare clic su Start, quindi su Pannello di controllo.

    In Visualizzazione classica, nel riquadro di spostamento, fare clic su Passa alla visualizzazione per categorie.

  2. Fare clic su Centro sicurezza PC, quindi in Gestisci impostazioni di protezione per fare clic su Windows Firewall.

  3. Accertarsi che la casella di controllo Non consentire eccezioni sia deselezionata.

  4. Fare clic sulla scheda Eccezioni e verificare che la casella di controllo Desktop remoto sia selezionata.

Causa: Non è possibile connettere il Mac alla rete.

Soluzione: Accertarsi che il computer Windows sia attivato e connesso alla rete.

Una connessione remota potrebbe essere interrotta a causa di problemi con la rete. Ad esempio, una connessione remota potrebbe non riuscire se la rete remota è inattiva o se il cavo Ethernet sul computer Windows è scollegato.

Causa: I criteri di rete stanno bloccando le connessioni.

Soluzione: Rivolgersi all'amministratore di rete in merito ai criteri di rete dell'organizzazione.

I criteri di protezione variano in base all'organizzazione. Ad esempio, se si desidera connettersi a un computer Windows su una rete che utilizza IPsec per consentire la protezione del traffico di rete, è necessaria un'eccezione limite per tale computer Windows.

Causa: Non è possibile rilevare il computer Windows.

Soluzione: Controllare che le impostazioni di rete sul Mac siano configurate correttamente per il tipo di connessione utilizzata.

Le impostazioni della connessione Ethernet o wireless sul Mac devono essere configurate per la rete alla quale si desidera connettersi.

Soluzione: Verificare che il nome computer immesso sia corretto.
Verifica del nome del computer in Windows Vista

  1. Dal computer Windows fare clic su Start , quindi su Pannello di controllo.

    In Visualizzazione classica, nel riquadro di spostamento, fare clic su Pagina iniziale Pannello di controllo.

  2. Fare clic su Sistema e manutenzione, quindi su Sistema.

  3. In Impostazioni relative a nome computer, dominio e gruppo di lavoro immettere il nome del computer o il come completo del computer se il computer Windows è membro di un dominio.

    Quando si apre Connessione desktop remoto su Mac, è necessario immettere questo nome.

Verifica del nome del computer in Windows XP

  1. Sul computer Windows fare clic su Start, fare clic con il pulsante destro del mouse su Computer locale, quindi scegliere Proprietà.

  2. Nella scheda Nome computer immettere il nome completo del computer.

    Quando si apre Connessione desktop remoto su Mac, è necessario immettere questo nome.

Soluzione: Utilizzare l'indirizzo IP se non è possibile connettersi utilizzando il nome del computer.

  1. Sul computer Windows fare clic su Start, scegliere Tutti i programmi, fare clic su Accessori, quindi su Prompt dei comandi.

  2. Al prompt dei comandi, digitare ipconfig, quindi premere INVIO.

  3. In Scheda Ethernet Connessione alla rete locale (LAN), annotare l'indirizzo IP del computer.

    Quando si apre Connessione desktop remoto su Mac, è possibile immettere l'indirizzo IP anziché il nome del computer. Su computer che eseguono Windows Vista, l'indirizzo IP è elencato come indirizzo IPv4.

Soluzione: Accertarsi che la connessione Internet sul Mac sia in funzione.

  • Aprire il browser Web (ad esempio Safari) e verificare che la pagina Web possa essere visualizzata.

Causa: L'identità del computer Windows non può essere verificata.

Soluzione: Controllare che la preferenza di protezione di Connessione desktop remoto sia compatibile con la versione di Windows in esecuzione sul computer Windows.

Se su un computer Windows è in esecuzione Windows XP o versione precedente oppure Windows Server 2003 Service Pack 1 (SP1) o versione precedente, non sarà possibile ottenere l'identità per la verifica. Se sul computer remoto è in esecuzione una di queste versioni di Windows e la preferenza di protezione sul Mac richiede l'autenticazione per la connessione, la connessione remota non potrà essere eseguita.

  1. Dal menu Connessione desktop remoto scegliere Preferenze, quindi fare clic su Protezione.

  2. Accertarsi che l'opzione Non connettere se l'autenticazione non riesce non sia selezionata.
