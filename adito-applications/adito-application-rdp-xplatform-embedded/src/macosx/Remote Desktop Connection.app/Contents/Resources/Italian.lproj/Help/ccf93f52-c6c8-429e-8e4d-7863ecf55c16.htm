Opzioni di Accesso universale

Connessione desktop remoto utilizza un numero di selezione delle funzioni di accesso universale di Mac OS X. Le funzioni di Accesso universale semplificano l'uso del computer in caso di problemi visivi o uditivi o in caso di difficoltà di movimento.

Per impostare le opzioni di accesso, scegliere Preferenze di Sistema dal menu Apple, quindi in Sistema fare clic su Accesso Universale. Nella tabella seguente sono elencate le funzioni di accesso universale di Mac OS X che è possibile utilizzare in Connessione desktop remoto.

Le funzioni di Mac OS X non funzionano sul desktop Windows dopo aver stabilito la sessione di desktop remoto. Per informazioni sulle funzioni di accessibilità disponibili in Windows, vedere la pagina Web Accesso facilitato Microsoft (informazioni in lingua inglese) all'indirizzo http://www.microsoft.com/enable.

VoiceOver

Consente di ascoltare la maggior parte dei comandi di menu, delle finestre di dialogo e di altri elementi in Connessione desktop remoto. Non funziona con il contenuto della finestra della sessione desktop remoto. Ad esempio, dopo aver stabilito la connessione a un computer Windows, VoiceOver non legge alcune elemento sul desktop Windows.
