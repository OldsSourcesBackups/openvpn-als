Connessione a più computer

Mostra tutto

Se si desidera connettersi a più computer Windows contemporaneamente, è possibile salvare un file di connessione per ciascun computer a cui ci si connette. Ciascun file di connessione può avere impostati set diversi di preferenze, come credenziali di accesso automatico, dimensioni schermo e opzioni di visualizzazione. È possibile connettersi rapidamente al computer desiderato aprendo il file di connessione.

Importante

Il computer Windows deve essere attivato e deve disporre di una connessione di rete. Non è possibile connettersi a un computer inattivo, sospeso o in standby. Pertanto, accertarsi che le opzioni di inattività, sospensione e standby sul computer Windows siano impostate su Mai. Per informazioni su come apportare queste modifiche, consultare la Guida di Windows. Accertarsi che il computer a cui si desidera eseguire la connessione esegua una versione di Windows in grado di supportare le connessioni remote. Per un elenco completo, vedere Compatibilità della versione Windows.

Passaggio 1: Connettersi al primo computer Windows

  1. Sul computer Macintosh aprire Connessione desktop remoto.

  2. Nella casella Computer digitare il nome del computer Windows a cui si desidera connettersi, quindi fare clic su Connetti.

  3. Digitare il nome utente, la password e il dominio per il computer Windows se necessario.

Passaggio 2: Salvare un file di connessione per il primo computer Windows

Selezionare le preferenze di connessione per il computer a cui si è connessi.

  1. Scegliere Preferenze dal menu Connessione desktop remoto.

  2. Selezionare le opzioni desiderate.

  3. Scegliere Salva con nome dal menu File.

  4. Nella casella Salva con nome digitare il nome del file di connessione.

  5. Scegliere Salva.

Suggerimento Per connettersi rapidamente a un computer Windows per il quale è stato salvato un file di connessione, scegliere Apri connessione salvata dal menu File, quindi fare clic sul file desiderato.

Passaggio 3: Connettersi a un computer Windows diverso

  1. Sulla barra dei menu di Connessione desktop remoto fare clic su File, quindi selezionare Nuova connessione.

  2. Nella casella Computer digitare il nome del computer Windows a cui si desidera connettersi, quindi fare clic su Connetti.

  3. Digitare il nome utente, la password e il dominio per il computer Windows se necessario.

    Sul desktop Mac sono quindi aperte due finestre di sessione, una per ciascun computer Windows a cui si è connessi. Per aprire ulteriori connessioni, ripetere questo passaggio.

Passaggio 4: Salvare un file di connessione per un computer Windows diverso

  1. Selezionare la finestra di sessione per il computer Windows aggiuntivo a cui si è stabilita la connessione.

  2. Scegliere Preferenze dal menu Connessione desktop remoto.

  3. Selezionare le opzioni desiderate.

  4. Scegliere Salva con nome dal menu File.

  5. Nella casella Salva con nome digitare il nome del file di connessione.

  6. Scegliere Salva.

  7. Ripetere questa procedura per ciascun computer Windows aggiuntivo a cui si è stabilita la connessione.

Suggerimento Per connettersi rapidamente a un computer Windows per il quale è stato salvato un file di connessione, scegliere Apri connessione salvata dal menu File, quindi fare clic sul file desiderato.
