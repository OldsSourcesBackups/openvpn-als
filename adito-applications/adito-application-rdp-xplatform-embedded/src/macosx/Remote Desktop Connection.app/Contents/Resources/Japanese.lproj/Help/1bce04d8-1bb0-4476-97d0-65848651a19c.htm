全画面表示モードで作業する

すべて表示

Windows ベースのコンピュータに初めて接続すると、リモート セッションは Mac のデスクトップのウインドウに表示されます。必要に応じて、このウインドウ モードでの表示を全画面表示モードに切り替えることができます。全画面表示モードに切り替えると、ウインドウが最大化され、Windows ベースのコンピュータのデスクトップのみが表示された状態になります。

必要に応じて、次の操作を行います。

全画面表示モードに切り替える
重要

全画面表示モードに切り替えると、Mac のメニュー バーと Dock が非表示になります。メニュー バーまたは Dock を表示するには、通常、メニュー バーや Dock が表示される画面の端をポイントします。後でウインドウ モードに戻るには、画面の上端にポインタを置いて RDC のメニュー バーが表示されるのを待ち、メニュー バーが表示されたら、 [表示] メニューをクリックし、 [全画面表示] のチェックをオフにします。

  • [表示] メニューの [全画面表示] をクリックします。

    ヒント 全画面表示モードで Dock をポイントすると、Dock により Windows タスク バーが隠れます。Dock (およびメニュー バー) が全画面表示モードで表示されたり、Dock によりタスク バーが隠れたりしないようにするには、Remote Desktop Connection の表示の環境設定を変更してください。

全画面表示モードでキーボード ショートカットを使用する

  • 必要に応じて、次の操作を行います。

    目的

    キー操作

    ウインドウ モードと全画面表示モードを切り替える

    ⌘+2

    画面の表示をウインドウ サイズに合わせて拡大/縮小するか、サイズを変更したときにスクロール バーを表示するかを切り替える

    ⌘+1

    Dock にリモート セッションを表示する

    ⌘+H

    Remote Desktop Connection を終了する

    ⌘+Q
