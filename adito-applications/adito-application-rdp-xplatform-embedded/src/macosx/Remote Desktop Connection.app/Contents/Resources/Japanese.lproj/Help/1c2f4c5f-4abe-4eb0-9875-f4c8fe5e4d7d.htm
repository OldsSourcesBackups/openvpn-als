セッションの速度を向上させる

アニメーションおよびグラフィックスを多用する機能がネットワーク接続で実行されると、リモート デスクトップ セッションのパフォーマンスが低下し、プログラムの実行速度が遅くなる可能性があります。セッションの開始後、Windows ベースのコンピュータで画面が変更されると、Macintosh コンピュータも更新されます。アニメーション処理では画面を更新し続ける必要があるため、パフォーマンスの低下につながります。これは、低速ネットワーク接続の場合に顕著です。たとえば、プログラムや Web 上でビデオやアニメーション画像を表示するときにこのような問題が発生します。

次に示すようなオプションがオンになっていると、セッションのパフォーマンスが低下することがあります。

  • デスクトップの背景を表示する

  • フォント スムージングを表示する

  • ドラッグ中にウインドウの内容を表示する

  • メニューおよびウインドウ アニメーションを表示する

  • テーマを表示する

パフォーマンスを向上させるには、これらのオプションのいくつかをオフにしてください。

ヒント 表示オプションをオフにするには、 [RDC] メニューの [環境設定] で [表示] をクリックし、目的のチェック ボックスをオフにします。
