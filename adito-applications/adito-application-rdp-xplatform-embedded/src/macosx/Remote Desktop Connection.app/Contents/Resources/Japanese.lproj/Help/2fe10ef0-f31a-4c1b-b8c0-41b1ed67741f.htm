マウスの右ボタンとして機能するキーの組み合わせを変更する

すべて表示

Mac にシングル ボタン マウスが接続されている場合は、キーを独自に組み合わせて、マウスの右ボタンと同じ機能を実現することができます。キーを押したままマウス ボタンをクリックすると、Windows ベースのコンピュータは、マウスの右ボタンがクリックされた、つまり "右クリックされた" と認識します。

メモ Remote Desktop Connection の既定の設定では、control+shift キーを押しながらクリックすると、右クリックと同じ動作になります。

  1. [RDC] メニューの [環境設定] をクリックし、 [キーボード] をクリックします。

  2. [マウスの右ボタン] の横にある [Mac] 列の既存のキーボード ショートカットをダブルクリックし、1 つまたは複数の修飾キーを押しながら (たとえば、⌘+shift キーを押しながら) シングルクリックします。
