コンピュータ間でファイルをコピーする

すべて表示

ファイルをコピーするには、Windows ベースのコンピュータ上で Mac のディスクまたはフォルダを開き、ファイルをコピーして Windows ベースのコンピュータにペーストします。ファイルを Mac から Windows ベースのコンピュータにドラッグすることはできません。

ステップ 1: Mac のディスクまたはフォルダを Windows ベースのコンピュータから利用できるようにする

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [ドライブ] をクリックし、ポップアップ メニューで必要なオプションを選択します。

ディスクまたはフォルダが利用可能になり、Windows ベースのコンピュータから Mac 上のファイルにアクセスできるようになります。

ステップ 2: コンピュータ間でファイルをコピーする

Mac のディスクまたはフォルダを Windows ベースのコンピュータから利用できるようにすると、コンピュータ間でファイルをコピーできます。

  1. [スタート] メニューの [コンピュータ] または [マイ コンピュータ] をクリックします。

    利用可能になった Mac のディスクまたはフォルダが、Windows ベースのコンピュータ上の他のすべてのディスクやフォルダと共に一覧に表示されます。

  2. コピー元のディスクまたはフォルダを開きます。

  3. [スタート] メニューの [コンピュータ] または [マイ コンピュータ] をクリックします。

  4. コピー先のディスクまたはフォルダを開きます。

    これで、2 つのウインドウが開いて、コピー元の場所とコピー先の場所が表示されている状態になります。

  5. コピー元のドキュメントをコピー先の場所までドラッグします。
