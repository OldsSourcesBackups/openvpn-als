接続を終了する

すべて表示

リモート セッションを終了するには、Windows ベースのコンピュータで接続を切断するか、ログオフします。切断すると接続は終了しますが、使用中の Windows ベースのプログラムやファイルは開いたまま引き続き実行されるので、後で再び接続して、同じプログラムやファイルを操作できます。ログオフするとセッションは終了します。セッション内部で実行されているすべてのプログラムは終了し、開いていたファイルの未保存の変更内容は失われます。

Windows Vista を実行しているコンピュータの接続を終了する

Windows ベースのコンピュータ上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

切断する

  • [スタート] ボタン () をクリックし、 [切断]  () をクリックします。

ログオフする

  • [スタート] ボタン () をクリックし、ロック アイコン () の横にある矢印をクリックして、 [ログオフ] をクリックします。

Windows XP を実行しているコンピュータの接続を終了する

Windows ベースのコンピュータ上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

切断する

  • [スタート] ボタンをクリックして [切断]  () をクリックし、もう一度 [切断] をクリックします。

ログオフする

  • [スタート] ボタンをクリックして [ログオフ] をクリックし、もう一度 [ログオフ] をクリックします。

Windows Server 2003 を実行しているコンピュータの接続を終了する

Windows ベースのコンピュータ上でプログラムを実行したまま接続を切断するには、[切断] を使用します。すべてのプログラムを終了して Windows からログオフするには、[ログオフ] を使用します。

切断する

  • リモート デスクトップのセッション ウインドウの をクリックし、 [切断] をクリックします。

ログオフする

  • [スタート] ボタンをクリックして [ログオフ] をクリックし、もう一度 [ログオフ] をクリックします。
