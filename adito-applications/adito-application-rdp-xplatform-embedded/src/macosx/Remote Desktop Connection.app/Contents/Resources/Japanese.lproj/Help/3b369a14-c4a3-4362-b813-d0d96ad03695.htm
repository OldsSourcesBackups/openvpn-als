Windows のユーザー名とパスワードを Mac OS のキーチェーンに追加する

すべて表示

ログイン情報を Mac OS のキーチェーンに保存しておくと、Windows ベースのコンピュータにすばやく接続できます。キーチェーンに追加した情報は接続ファイルリモート接続に関するすべての情報を格納するファイル。たとえば、セッション ウインドウの画面サイズや Windows ベースのコンピュータから使用できるようにする Mac ドライブなどの設定は、接続ファイルに保存できます。Windows ベースの複数のコンピュータに接続する場合は、接続するコンピュータごとに接続ファイルを保存することができます。に保存され、接続のたびにログイン情報を入力する必要がなくなります。

メモ パスワードは、接続ファイルには保存されません。Mac OS のキーチェーンにのみ保存されます。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [ログイン] をクリックします。

  3. Windows ベースのコンピュータのユーザー名、パスワード、ドメインを入力し、 [ユーザー情報をキーチェーンに追加する] チェック ボックスをオンにします。

    ヒント ユーザー情報の一部をキーチェーンに保存することもできます。たとえば、ユーザー名とドメインをキーチェーンに保存すると、パスワードを入力するだけでログインできるようになります。
