キーボード ショートカットを変更する

すべて表示

Remote Desktop Connection では、Windows ベースのコンピュータでの操作に対応する Mac 用のキーボード ショートカットが用意されています。このキーボード ショートカットは、無効にすることも、既定の割り当てを別のキーの組み合わせに変更することもできます。たとえば、使用しないキーボード ショートカットを無効にして、それと同じキーの組み合わせを他の操作に割り当てることができます。

必要に応じて、次の操作を行います。

キーボード ショートカットを無効にする

  1. [RDC] メニューの [環境設定] をクリックし、 [キーボード] をクリックします。

  2. 無効にするキーボード ショートカットの横にあるチェック ボックスをオフにします。

キーボード ショートカットを変更する

  1. [RDC] メニューの [環境設定] をクリックし、 [キーボード] をクリックします。

  2. [Mac] 列で、変更するキーボード ショートカットをダブルクリックし、代わりに使用するキーの組み合わせを押します。

    メモ キーの新しい組み合わせは、control+F10 のように、1 つの修飾キーと、その他の 1 つ以上のキーを組み合わせたものにする必要があります。
