Windows ベースのコンピュータでリモート アクセスを許可する

すべて表示

Remote Desktop Connection を使用して Windows ベースのコンピュータに接続するには、リモート接続ができるように Windows ベースのコンピュータを設定しておく必要があります。

重要

Windows ベースのコンピュータは、電源がオンで、ネットワーク接続が確立された状態である必要があります。スリープ状態、休止状態、スタンバイ状態のコンピュータに接続することはできません。したがって、Windows ベースのコンピュータのスリープ、休止、およびスタンバイの設定は必ず [なし] に設定してください。スリープ、休止、およびスタンバイの設定の変更方法については、Windows のヘルプを参照してください。接続先のコンピュータで実行している Windows が、リモート接続をサポートするバージョンであることも確認してください。バージョンの一覧については、「Windows バージョンの互換性」を参照してください。

必要に応じて、次の操作を行います。

Windows Vista を実行しているコンピュータのリモート接続を許可する
重要

リモート デスクトップの設定を変更するには、Windows ベースのコンピュータの管理者権限が必要です。

  1. Windows ベースのコンピュータで、 [スタート] ボタン () をクリックし、 [コントロール パネル] をクリックします。

    クラシック表示の場合は、ナビゲーション ウインドウの [コントロール パネル ホーム] をクリックします。

  2. [システムとメンテナンス] をクリックし、 [システム] をクリックします。

  3. ナビゲーション ウインドウで [リモートの設定] をクリックし、 [リモート デスクトップ] で必要なオプションをクリックします。

    管理者のパスワードを求めるメッセージが表示されたらパスワードを入力し、その後の確認メッセージに従って、もう一度パスワードを入力します。

    メモ ネットワーク レベル認証によって、Mac からリモート接続を開始する際に Windows のログイン情報の入力が必要になるため、セキュリティが向上します。ネットワーク レベル認証の詳細については、Windows Vista のヘルプを参照してください。

  4. [ユーザーの選択] をクリックします。

  5. 必要に応じて、次の操作を行います。

    目的

    操作

    アクセス許可を確認する

    自分のユーザー アカウントがボックスの一覧に表示されていることを確認します。コンピュータの管理者のアカウントは自動的に一覧に表示されます。

    他のユーザー アカウントを追加する

    [追加] をクリックし、ボックス内に、追加するユーザー アカウント名を入力します。

Windows XP を実行しているコンピュータのリモート接続を許可する
重要

リモート デスクトップの設定を変更するには、Windows ベースのコンピュータの管理者権限が必要です。

  1. Windows ベースのコンピュータで、 [スタート] ボタンをクリックし、 [コントロール パネル] をクリックします。

    クラシック表示の場合は、ナビゲーション ウインドウの [カテゴリの表示に切り替える] をクリックします。

  2. [パフォーマンスとメンテナンス] をクリックし、 [システム] をクリックします。

  3. [リモート] タブで、 [このコンピュータにユーザーがリモートで接続することを許可する] チェック ボックスをオンにします。

  4. [リモート ユーザーの選択] をクリックします。

  5. 必要に応じて、次の操作を行います。

    目的

    操作

    アクセス許可を確認する

    自分のユーザー アカウントがボックスの一覧に表示されていることを確認します。コンピュータの管理者のアカウントは自動的に一覧に表示されます。

    他のユーザー アカウントを追加する

    [追加] をクリックし、ボックス内に、追加するユーザー アカウント名を入力します。
