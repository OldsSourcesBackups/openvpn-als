接続ファイルを作成および変更する

すべて表示

Remote Desktop Connection では、接続の環境設定は既定の接続ファイルに保存されます。接続先の Windows ベースのコンピュータごとにコンピュータ名と接続の設定を保存しておくと、接続にかかる時間を短縮することができます。たとえば、ある Windows ベースのコンピュータに接続するときはすべての Mac ディスクを共有し、別の Windows ベースのコンピュータに接続するときは 1 つの Mac フォルダのみを共有する、のような設定が可能です。同じ Windows ベースのコンピュータについて別の接続ファイルを保存することもできます。

必要に応じて、次の操作を行います。

接続ファイルを作成する

  1. [RDC] メニューの [環境設定] をクリックします。

    メモ 環境設定を繰り返し行う場合は、その都度ログイン ウインドウにユーザー情報を入力する必要はありません。

  2. 接続ごとに、画面のサイズ、自動ログイン情報、セキュリティ オプションなどの接続の環境設定を選択します。

  3. [ファイル] メニューの [別名で保存] をクリックします。

  4. 接続ファイルの名前を入力し、 [保存] をクリックします。

    接続が確立されると、ここで選択したオプションがすべてのリモート デスクトップ接続の既定の環境設定になります。

ヒント 接続ファイルに保存した Windows ベースのコンピュータにすばやく接続するには、 [ファイル] メニューの [保存した接続を開く] をクリックし、目的のファイルをクリックします。

接続ファイルを変更する

  1. Remote Desktop Connection で、 [ファイル] メニューの [接続の編集] をクリックします。

  2. 目的の接続ファイルを選択し、 [開く] をクリックします。

  3. 必要な変更を加えます。

  4. [ファイル] メニューの [保存] をクリックします。

    接続が確立されると、ここで選択したオプションがすべてのリモート デスクトップ接続の既定の環境設定になります。

    ヒント セッションの途中でも、 [RDC] メニューの [環境設定] をクリックして、接続の環境設定を変更することができます。この変更は、次回の接続時に有効になります。
