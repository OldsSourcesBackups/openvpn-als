サーバーのコンソール セッションに接続する

すべて表示
重要

Windows Server を実行しているリモート サーバー上のコンソール セッションにログインするには、Windows ベースのコンピュータの管理者権限が必要です。

  1. Remote Desktop Connection を起動します。

  2. [コンピュータ] ボックスで、接続先の Windows ベースのコンピュータの名前または IP アドレスに続けて「/console」と入力します (例 : FabricamFileServer/console)。

  3. [接続] をクリックします。

  4. ユーザー名、パスワード、およびドメインを必要に応じて入力します。

メモ このログイン方法は、Windows Server を実行しているサーバーに接続する場合にのみ使用できます。
