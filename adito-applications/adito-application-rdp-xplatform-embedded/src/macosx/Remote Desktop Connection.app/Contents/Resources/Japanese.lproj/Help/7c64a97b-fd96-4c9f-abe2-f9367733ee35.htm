Mac のハード ディスクにアクセスできない

すべて表示

原因 : Mac のハード ディスクが Windows ベースのコンピュータと共有されていません。

対処方法 : Windows ベースのコンピュータから利用する Mac のディスクまたはフォルダを選択します。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [ドライブ] をクリックし、ポップアップ メニューで必要なオプションを選択します。

    ディスクまたはフォルダが利用可能になり、Windows ベースのコンピュータから Mac 上のファイルにアクセスできるようになります。

  3. Windows ベースのコンピュータで、 [スタート] ボタンをクリックし、 [コンピュータ] または [マイ コンピュータ] をクリックします。

    利用可能になった Mac のディスクまたはフォルダが、Windows ベースのコンピュータ上の他のすべてのディスクやフォルダと共に一覧に表示されます。

原因 : ターミナル サービスのグループ ポリシーでディスクの共有が許可されていません。

対処方法 : ターミナル サービスのグループ ポリシーの詳細については、ネットワーク管理者に問い合わせてください。

グループ ポリシーは、Remote Desktop Connection のどの環境設定よりも優先されます。 ターミナル サービスのグループ ポリシーでディスクの共有が禁止されている場合、Windows ベースのコンピュータで Mac のディスクを表示することはできません。
