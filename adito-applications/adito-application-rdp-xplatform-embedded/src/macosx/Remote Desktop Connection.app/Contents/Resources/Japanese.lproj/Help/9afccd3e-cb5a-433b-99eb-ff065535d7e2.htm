オンライン ヘルプが利用できない

すべて表示

原因 : ネットワークまたはインターネット接続が利用できなくなっています。

対処方法 : オフライン ヘルプを使用します。

  • ヘルプ ウインドウの左下の矢印をクリックして、 [オフライン ヘルプ] をクリックします。

    ヒント インターネット接続が復旧したら、ヘルプ ウインドウの左下の矢印をクリックして、 [オンライン ヘルプ] をクリックします。

対処方法 : ネットワークまたはインターネットに接続していることを確認します。

ネットワーク接続の状態を確認する方法については、Mac OS のヘルプを参照してください。また、ネットワーク ケーブルが差し込まれていることや、他のネットワーク機器 (ワイヤレス モデムなど) に異常がないことも確認してください。ネットワーク管理者またはインターネット サービス プロバイダ (ISP)電子メール、チャット ルーム、World Wide Web などを利用するためのインターネット サービスを提供する企業。各国に多数のアクセス拠点を持つ ISP もあれば、特定の地域や国に限定されたサービスを提供する ISP もあります。 に問い合わせて、ネットワークやインターネット接続が使用可能かどうかを確認してください。
