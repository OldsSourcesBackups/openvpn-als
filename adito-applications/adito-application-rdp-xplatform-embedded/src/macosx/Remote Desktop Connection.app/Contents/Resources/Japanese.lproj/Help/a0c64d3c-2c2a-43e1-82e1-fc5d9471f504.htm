見つかりませんでしたか?

より適切な検索結果を得るには、次のヒントを参考にしてください。

  • 具体的でわかりやすい語句を使用します。

  • 似た意味の別な語句を試します。検索するコンテンツで使用されていそうな語句を選んでください。指定した語句が具体的であるほど、期待したものに近い検索結果が得られます。実行したい作業の内容を的確に表す語句に絞ってください。

  • 指定する語句を増やします。複数の語句で検索することにより、より正確な検索結果が得られる場合があります。

  • 検索する語句を正しく入力します。

  • 他の表現を試します。

たとえば、Word 文書に改ページを手動で挿入する方法について検索する場合は、検索ボックスに「改ページ」と入力するだけでなく、次の例のように具体的な語句を指定します。

  • 改ページを挿入する
