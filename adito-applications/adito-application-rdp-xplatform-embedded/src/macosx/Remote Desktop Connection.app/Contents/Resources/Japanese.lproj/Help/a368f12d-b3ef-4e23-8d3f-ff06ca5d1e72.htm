Mac メニュー バーが表示されない

すべて表示

原因 : リモート デスクトップ セッションが全画面表示で実行されています。

対処方法 : 通常、メニュー バーが表示される画面の端をポイントします。

全画面表示では、使用できる画面スペースを最大にするためにメニュー バーは表示されません。

原因 : 表示の環境設定の全画面表示に関するオプションが、メニュー バーと Dock を表示しないように設定されています。

対処方法 : 表示の環境設定の全画面表示に関するオプションをオフにします。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [表示] をクリックし、 [全画面表示でメニュー バーと Dock を非表示にする] チェック ボックスをオフにします。
