複数のコンピュータに接続する

すべて表示

接続先の Windows ベースのコンピュータごとに接続ファイルを保存すると、複数のコンピュータに同時に接続できます。接続ファイルごとに、自動ログイン資格情報、画面のサイズ、表示オプションなどのさまざまな環境設定を指定でき、この接続ファイルを開くことで、対象のコンピュータにすばやく接続することができます。

重要

Windows ベースのコンピュータは、電源がオンで、ネットワーク接続が確立された状態である必要があります。スリープ状態、休止状態、スタンバイ状態のコンピュータに接続することはできません。したがって、Windows ベースのコンピュータのスリープ、休止、およびスタンバイの設定は必ず [なし] に設定してください。スリープ、休止、およびスタンバイの設定の変更方法については、Windows のヘルプを参照してください。接続先のコンピュータで実行している Windows が、リモート接続をサポートするバージョンであることも確認してください。バージョンの一覧については、「Windows バージョンの互換性」を参照してください。

ステップ 1: 1 台目の Windows ベースのコンピュータに接続する

  1. Macintosh コンピュータで、Remote Desktop Connection を起動します。

  2. [コンピュータ] ボックスに接続先の Windows ベースのコンピュータの名前を入力し、 [接続] をクリックします。

  3. 必要に応じて、Windows ベースのコンピュータのユーザー名、パスワード、およびドメインを入力します。

ステップ 2: 1 台目の Windows ベースのコンピュータの接続ファイルを保存する

接続中のコンピュータに使用している、接続の環境設定を選択します。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. 必要なオプションを選択します。

  3. [ファイル] メニューの [別名で保存] をクリックします。

  4. [名前] ボックスに、接続ファイル名を入力します。

  5. [保存] をクリックします。

ヒント 接続ファイルに保存した Windows ベースのコンピュータにすばやく接続するには、 [ファイル] メニューの [保存した接続を開く] をクリックし、目的のファイルをクリックします。

ステップ 3: 他の Windows ベースのコンピュータに接続する

  1. Remote Desktop Connection のメニュー バーで [ファイル] をクリックし、 [新しい接続] をクリックします。

  2. [コンピュータ] ボックスに接続先の Windows ベースのコンピュータの名前を入力し、 [接続] をクリックします。

  3. 必要に応じて、Windows ベースのコンピュータのユーザー名、パスワード、およびドメインを入力します。

    Mac デスクトップには、接続先の Windows ベースのコンピュータごとに 1 つずつ、合計 2 つのセッション ウインドウが表示されます。さらに他の接続を開くには、この操作を繰り返します。

ステップ 4: 他の Windows ベースのコンピュータの接続ファイルを保存する

  1. 対象となる接続先の Windows ベースのコンピュータのセッション ウインドウを選択します。

  2. [RDC] メニューの [環境設定] をクリックします。

  3. 必要なオプションを選択します。

  4. [ファイル] メニューの [別名で保存] をクリックします。

  5. [名前] ボックスに、接続ファイル名を入力します。

  6. [保存] をクリックします。

  7. 対象となる接続先の Windows ベースのコンピュータごとに、この操作を繰り返します。

ヒント 接続ファイルに保存した Windows ベースのコンピュータにすばやく接続するには、 [ファイル] メニューの [保存した接続を開く] をクリックし、目的のファイルをクリックします。
