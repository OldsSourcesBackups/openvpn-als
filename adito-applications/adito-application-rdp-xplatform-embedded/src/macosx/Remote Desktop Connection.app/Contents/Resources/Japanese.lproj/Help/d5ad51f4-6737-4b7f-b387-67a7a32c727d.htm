音声が再生されない

すべて表示

原因 : サウンドの環境設定がオフになっています。

対処方法 : サウンドを再生するオプションを選択します。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [サウンド] をクリックし、必要なオプションをクリックします。

原因 : ターミナル サービスのグループ ポリシーで、Windows ベースのコンピュータからのサウンドのリダイレクトが許可されていません。

対処方法 : ターミナル サービスのグループ ポリシーの詳細については、ネットワーク管理者に問い合わせてください。

グループ ポリシーは、Remote Desktop Connection のどの環境設定よりも優先されます。 ターミナル サービスのグループ ポリシーでサウンドのリダイレクトが禁止されている場合、Windows ベースまたは Mac のいずれのコンピュータでもサウンドは再生されません。

原因 : スピーカーの音量 (ボリューム) が小さいか、消音 (ミュート) に設定されています。

対処方法 : Mac コンピュータと Windows ベースのコンピュータの両方のサウンド設定を確認します。

いずれかのコンピュータで音量がオフになっている場合は、サウンドは再生されません。
