ドキュメントをプリントする

すべて表示

Windows ベースのコンピュータにあるドキュメントを、Mac に接続されているプリンタでプリントできます。そのためには、まずプリンタを Windows ベースのコンピュータと共有する必要があります。

プリンタを利用できるようにすると、Windows ベースのコンピュータからプリンタにアクセスできます。接続ファイルリモート接続に関するすべての情報を格納するファイル。たとえば、セッション ウインドウの画面サイズや Windows ベースのコンピュータから使用できるようにする Mac ドライブなどの設定は、接続ファイルに保存できます。Windows ベースの複数のコンピュータに接続する場合は、接続するコンピュータごとに接続ファイルを保存することができます。を保存しておくと、後のリモート セッションでその接続ファイルを使ってログインしたときに、プリンタはそのまま利用できます。

必要に応じて、次の操作を行います。

プリンタを Windows ベースのコンピュータから利用できるようにする

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [プリンタ] をクリックし、 [Mac に接続されたプリンタを使用する] チェック ボックスをオンにします。

  3. 目的のプリンタを選択します。

    ヒント 

    • プリント時にプリンタを選択する場合は、 [すべてのプリンタ] をクリックします。

    • プリンタへのアクセス情報を含む接続ファイルを保存するには、 [ファイル] メニューの [別名で保存] をクリックし、接続ファイル名を入力します。

Windows ベースのドキュメントを Mac に接続されているプリンタでプリントする

  1. Windows デスクトップで、プリントするドキュメントを開いてプリントを実行します。

  2. 出力先のプリンタを選択し、 [OK] をクリックします。
