Mac に接続されているプリンタでプリントできない

すべて表示

原因 : Mac に接続されているプリンタが、Windows ベースのコンピュータから利用できないようになっています。

対処方法 : プリンタを利用できるように、プリンタの環境設定を行います。

  1. [RDC] メニューの [環境設定] をクリックします。

  2. [プリンタ] をクリックし、 [Mac に接続されたプリンタを使用する] チェック ボックスをオンにします。

  3. 目的のプリンタを選択します。

    ヒント プリント時にプリンタを選択する場合は、 [すべてのプリンタ] をクリックします。

原因 : ターミナル サービスのグループ ポリシーでプリンタの共有が許可されていません。

対処方法 : ターミナル サービスのグループ ポリシーの詳細については、ネットワーク管理者に問い合わせてください。

グループ ポリシーは、Remote Desktop Connection のどの環境設定よりも優先されます。 グループ ポリシーでプリンタの共有が禁止されている場合、Windows ベースのコンピュータのドキュメントを Mac に接続されているプリンタでプリントできません。

原因 : プリンタの接続ケーブルが外れているか、Mac に接続されているプリンタの電源がオフになっています。

対処方法 : プリンタの状態を確認します。

  • プリンタの接続ケーブルをしっかりと差し込み、プリンタの電源をオンにします。

ヒント プリンタが Mac に接続されていることを確認するには、 [アップル] メニューの [システム環境設定] をクリックし、 [ハードウェア] の [プリントとファクス] をクリックします。
