Novedades en Conexión a Escritorio remoto

Cliente de conexión a Escritorio remoto de Microsoft para Mac 2 incluye un nuevo diseño de interfaz de usuario, opciones de personalización mejoradas y la posibilidad de conectarse a varios equipos basados en Windows al mismo tiempo. En la siguiente tabla se muestran las mejoras más recientes de Conexión a Escritorio remoto.

Característica

Descripción

Binario universal

Se ejecuta en Mac basados en Intel y basados en PowerPC.

Interfaz de usuario y barra de menús

Simplifica la ventana de conexión y expande la barra de menús para incluir menús de tareas comunes (Archivo, Edición, Ver y Ayuda). Ahora las preferencias de la aplicación se encuentran en el menú RDC.

Protocolo de escritorio remoto 6.0

Mejora el rendimiento y la compatibilidad con Windows Vista.

Varias sesiones

Conecta con varios equipos cuando guarda un archivo de conexión para cada equipo. Utilice el nuevo menú Archivo para abrir conexiones nuevas, guardadas y abiertas anteriormente y para gestionar todos los cambios en los archivos de conexión.

Mejor personalización que en Conexión a Escritorio remoto 1.0

Acepta los cambios en las preferencias de la aplicación, como métodos abreviados del teclado, durante una sesión de Escritorio remoto. Los cambios se aplicarán la próxima vez que se conecte.

Reconexión automática

Restablece una sesión remota cuando se pierde la conexión de red.

Opciones de cambio de tamaño de pantalla

Ejecuta la sesión de Escritorio remoto en una ventana redimensionable o en el modo de pantalla completa.

Impresión

Imprime los documentos basados en Windows en cualquier impresora que esté conectada al equipo Mac.

Autenticación a nivel de red

Verifica la identidad del equipo basado en Windows antes de establecer una conexión de Escritorio remoto. Puede seleccionar esta opción al conectarse a un equipo que ejecuta Windows Vista. La Autenticación a nivel de red es más segura que las opciones de autenticación de versiones anteriores de Windows.

Resolución panorámica

Proporciona una configuración para mejorar la resolución en pantallas panorámicas.

Sesiones de consola de servidor

Acelera el inicio de sesión cuando se especifica "console" como parte de nombre del equipo.
