Limitar una sesión a una única aplicación basada en Windows

Mostrar todo

Si trabaja con frecuencia en una aplicación específica en un servidor que ejecuta Windows Server, puede ahorrar tiempo si inicia la sesión de Escritorio remoto únicamente con esa aplicación. Los administradores de red también pueden limitar los equipos cliente a una sola aplicación cuando se conecten a un servidor de Windows. El escritorio de Windows no es visible o no está disponible cuando se selecciona esta preferencia.

  1. En el menú RDC, haga clic en Preferencias y, a continuación, en Aplicaciones.

  2. Active la casilla de verificación.

  3. En el cuadro Nombre de archivo y ruta de acceso de aplicación, escriba la ruta y el nombre de archivo de la aplicación que desee iniciar.

    También puede escribir la ruta del directorio de trabajoLa carpeta en la que se instala y ejecuta una aplicación. La mayoría de las aplicaciones instaladas se diseñan con un directorio de trabajo fijo que no se tiene que cambiar. Algunas aplicaciones, programas de utilidad y determinados archivos por lotes y scripts se diseñan para que se ejecuten en una carpeta distinta de aquélla en la que se encuentra la aplicación. de la aplicación, si es necesario.

Sugerencia Para cerrar la conexión, salga de la aplicación o desconéctese del equipo basado en Windows.
