Opciones de acceso universal

Conexión a Escritorio remoto utiliza un número selecto de funciones de acceso universal de Mac OS X. Las funciones de acceso universal facilitan el uso del equipo cuando el usuario tiene problemas de audición o una destreza limitada.

Para establecer las opciones de acceso, en el menú Apple, haga clic en Preferencias del Sistema y, a continuación, en Sistema, haga clic en Acceso Universal. En la siguiente tabla se muestran las funciones de acceso universal de Mac OS X que puede utilizar en Conexión a Escritorio remoto.

Las funciones de Mac OS X no funcionan en el escritorio de Windows una vez se ha establecido la sesión de Escritorio remoto. Para obtener más información sobre las funciones de accesibilidad disponibles en Windows, consulte la página Web de Accesibilidad de Microsoft en http://www.microsoft.com/enable.

VoiceOver

Se oyen la mayoría de los comandos de menú, cuadros de diálogo y otros elementos de Conexión a Escritorio remoto. No funciona con el contenido de la ventana de sesión de Escritorio remoto. Por ejemplo, cuando se conecte a un equipo basado en Windows, VoiceOver no leerá ningún elemento del escritorio de Windows.

Acceso total del teclado

Utilice el teclado para desplazarse por los controles en lugar de usar el mouse (ratón).

Nota Las descripciones de las fichas hacen referencia a la disposición del teclado de EE. UU. Las teclas de otras disposiciones de teclado puede que no se correspondan con las teclas de un teclado de EE. UU. Los métodos abreviados del teclado para equipos portátiles también pueden ser diferentes.

Para obtener los métodos abreviados del teclado, consulte la Ayuda de Mac.
