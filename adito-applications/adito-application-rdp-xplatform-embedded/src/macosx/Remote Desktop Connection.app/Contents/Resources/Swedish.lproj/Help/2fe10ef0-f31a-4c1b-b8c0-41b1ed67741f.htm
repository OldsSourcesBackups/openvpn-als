Ändra vilka tangenter som ska fungera som höger musknapp

Visa alla

Om du har en mus med en enda knapp ansluten till din Mac så kan du skapa din egen kombination av tangenter som fungerar som höger musknapp. När du håller ned tangenterna och klickar på musknappen känner den Windows-baserade datorn igen detta som ett klick med höger musknapp (högerklick).

Obs! Som standard använder Anslutning till fjärrskrivbord CONTROL+SKIFT+ klick som högerklicksfunktion.

  1. Klicka på Inställningar och klicka sedan på Tangentbord på menyn RDC.

  2. Bredvid den sekundära musknappen högerklickar du på det befintliga kortkommandot i Mac-kolumnen och trycker sedan på eller eller flera låstangenter och ett enkelklick, till exempel KOMMANDO+SKIFT+klick.
