Lägg till ditt användarnamn och lösenord för Windows till Mac OS-nyckelringen

Visa alla

Du kan ansluta till en Windows-baserad dator snabbare genom att spara inloggningsinformationen i Mac OS-nyckelringen. Information som läggs till i nyckelringen sparas i anslutningsfilEn fil som innehåller all information som behövs för en fjärranslutning. Du kan till exempel ange skärmstorleken för sessionsfönstret eller dela Mac-drivrutiner med den Windows-baserade datorn genom attt spara de inställningarna i en anslutningsfil. Om du ansluter till flera Windows-baserade datorer kan du spara en ny anslutningsfil för varje dator. så att du inte måste ange det varje gång du ansluter.

Obs! Lösenord sparas bara i Mac OS-nyckelringen och inte i anslutningsfilen.

  1. Klicka på Inställningar på RDC-menyn.

  2. Klicka på Logga in.

  3. Ange användarnamn, lösenord och domän för den Windows-baserade datorn och markera sedan kryssrutan Lägg till användarinformation i din nyckelring.

    Tips! Du kan även spara ofullständig användarinformation i din nyckelring. Du kan exempelvis spara användarnamn och domän och sedan logga in genom att bara ange lösenordet.
