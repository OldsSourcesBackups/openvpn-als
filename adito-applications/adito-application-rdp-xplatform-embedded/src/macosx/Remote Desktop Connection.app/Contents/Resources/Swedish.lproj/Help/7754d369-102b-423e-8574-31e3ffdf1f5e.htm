Tangentbordsgenvägar

Du kan snabbt utföra åtgärder med hjälp av tangentbordsgenvägar, d.v.s. en eller flera tangenter som du trycker på på tangentbordet för att utföra en åtgärd.

Beskrivningarna av tangentbordsgenvägarna gäller för den amerikanska tangentbordslayouten. Tangenterna på tangentbord med andra layouter kanske inte stämmer med tangenterna på ett amerikanskt tangentbord. Tangentbordsgenvägar på bärbara datorer kan också skilja sig från dessa.

Redigering och formatering

När du arbetar med Windows-skrivbordet kan du utföra alla uppgifter som du är van vid genom att använda motsvarande kortkommandon för Windows. Om du till exempel vill kopiera text från ett dokument på ditt Windows-skrivbord kan du trycka på CTRL+C eller KOMMANDO+C. Om du vill klistra in innehållet i ett dokument på din Mac-dator trycker du på KOMMANDO+V.
