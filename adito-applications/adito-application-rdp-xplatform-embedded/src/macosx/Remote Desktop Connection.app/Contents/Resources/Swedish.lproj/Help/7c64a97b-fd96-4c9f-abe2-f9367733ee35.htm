Jag kommer inte åt mina Mac-hårddiskar

Visa alla

Orsak: Mac-hårddisken delas inte med den Windows-baserade datorn.

Lösning: Markera den Mac-disk eller mapp som du vill göra tillgänglig för den Windows-baserade datorn.

  1. Klicka på Inställningar på RDC-menyn.

  2. Klicka på Enheter och välj sedan det alternativ du vill använda på popup-menyn.

    Din enhet eller mapp är nu tillgänglig så att du kan få åtkomst till filer som finns på din Mac från din dator som kör Windows.

  3. Klicka på Start och klicka sedan på Dator eller Den här datorn på den Windows-baserade datorn.

    Mac-enheten eller mappen som du har gjort tillgänglig finns med tillsammans med alla andra enheter och mappar på din dator som kör Windows.

Orsak: Grupprincipen för Terminal Services tillåter inte delning av diskar.

Lösning: Kontakta nätverksadministratören om din grupprincip för Terminal Services.

Grupprincipen åsidosätter alla inställningar som du gör i Anslutning till fjärrskrivbord.Om grupprincipen för Terminal Services förhindrar delning av diskar kan du inte se dina Mac-diskar på den Windows-baserade datorn.
