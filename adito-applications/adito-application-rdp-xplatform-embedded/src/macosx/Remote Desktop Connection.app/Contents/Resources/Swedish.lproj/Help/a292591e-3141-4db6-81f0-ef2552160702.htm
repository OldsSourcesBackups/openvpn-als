Begränsa en session till ett enda Windows-baserat program

Visa alla

Om du ofta arbetar med ett specifikt program på en server som kör Windows Server kan du spara tid genom att starta fjärrskrivbordssessionen med bara det programmet. Nätverksadministratörer kan också begränsa klientdatorer till ett enda program när de är anslutna till en Windows-server. Windows-skrivbordet är inte synligt eller tillgängligt när den här inställningen har valts.

  1. Klicka på Inställningar och klicka sedan på Program på menyn RDC.

  2. Markera kryssrutan.

  3. Ange sökväg och filnamn för det program du vill starta i rutan Programmets sökväg och filnamn.

    Du kan också ange sökvägen till arbetskatalogDen mapp som ett program installeras och körs i. De flesta program som installeras har en fast arbetskatalog som inte behöver ändras. Vissa äldre program, hjälpmedelsprogram och vissa skript och batchfiler körs i en annan mapp än den där programmet finns. för programmet om det behövs.

Tips! För att stänga anslutningen avslutar du programmet eller kopplar från den Windows-baserade datorn.
