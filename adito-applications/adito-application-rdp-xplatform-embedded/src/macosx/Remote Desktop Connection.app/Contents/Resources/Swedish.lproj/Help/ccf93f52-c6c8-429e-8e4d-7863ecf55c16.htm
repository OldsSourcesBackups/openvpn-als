Universella åtkomstalternativ

Anslutning till fjärrskrivbord har några utvalda universella åtkomstalternativ från Mac OS X. Med universella åtkomstalternativ kan det bli lättare att använda din dator om du har svårt för att se eller höra eller om du har nedsatt rörlighet i händerna.

Klicka på Systeminställningar på Apple-menyn och klicka sedan på Universell åtkomst under System om du vill ange åtkomstalternativ. I följande tabell visas de universella åtkomstfunktionerna i Mac OS X som du kan använda i Anslutning till fjärrskrivbord.

Mac OS X-funktionerna fungerar inte på Windows-skrivbordet när fjärrskrivbordssessionen har upprättats. Om du vill veta mer om de hjälpmedelsfunktioner som finns i Windows går du till Microsoft-hjälpmedel-webbsidan på http://www.microsoft.com/enable.

VoiceOver

Lyssna på de flesta menykommandon, dialogrutor och andra element i Anslutning till fjärrskrivbord. Fungerar inte med innehållet i fjärrskrivbordsfönstret. När du har anslutit till en Windows-baserad dator läser funktionen VoiceOver till exempel inte några element på Windows-skrivbordet.

Fullständig åtkomst med tangentbordet

Använd tangentbordet för att flytta mellan kontroller i stället för att använda musen.
