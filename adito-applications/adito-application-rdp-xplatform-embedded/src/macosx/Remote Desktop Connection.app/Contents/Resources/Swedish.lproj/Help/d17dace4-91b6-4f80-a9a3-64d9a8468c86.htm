Anslut till flera datorer

Visa alla

Om du vill ansluta till mer än en Windows-baserad dator i taget kan du spara anslutningsfilen för varje dator som du ansluter till. Varje anslutningsfil kan ha olika inställningar som automatiska inloggningsuppgifter, skärmstorlek och visningsalternativ. Du kan snabbt ansluta till den dator du vill använda genom att öppna dess anslutningsfil.

Viktigt!

Den Windows-baserade datorn måste vara på, och det måste finnas en nätverksanslutning. Du kan inte ansluta till en dator som är i viloläge eller vänteläge. Se till att inställningarna för viloläge och vänteläge på den Windows-baserade datorn är inställda på Aldrig. Om du vill veta hur du gör de här ändringarna läser du i hjälpen för Windows. Kontrollera att den dator som du vill ansluta till kör en version av Windows som stödjer fjärranslutningar. En fullständig lista finns på Kompatibilitet med Windows-versioner.

Steg 1: Anslut till den första Windows-baserade datorn

  1. Öppna Anslutning till fjärrskrivbord på din Macintosh-dator.

  2. Skriv namnet på den Windows-baserade dator som du vill ansluta till i rutan Dator och klicka sedan på Anslut.

  3. Ange användarnamn, lösenord och domän för den Windows-baserade datorn om de krävs.

Steg 2: Spara en anslutningsfil för den första Windows-baserade datorn

Välj anslutningsinställningar för den dator som du är ansluten till.

  1. Klicka på Inställningar på RDC-menyn.

  2. Välj de alternativ som du vill använda.

  3. Klicka på Spara som på Arkiv-menyn.

  4. Skriv namnet på din anslutningsfil i rutan Spara som.

  5. Klicka på Spara.

Tips! Klicka på Öppna en sparad anslutning på Arkiv-menyn och klicka sedan på filen du vill använda om du snabbt vill ansluta till en Windows-baserad dator som du sparat en anslutningsfil för.

Steg 3: Anslut till en annan Windows-baserad dator

  1. I menyfältet för Anslutning till fjärrskrivbord klickar du på Arkiv och klickar sedan på Ny anslutning.

  2. Skriv namnet på den Windows-baserade dator som du vill ansluta till i rutan Dator och klicka sedan på Anslut.

  3. Ange användarnamn, lösenord och domän för den Windows-baserade datorn om de krävs.

    Det finns nu två sessionsfönster på ditt Mac-skrivbord: ett för varje Windows-baserad dator som du är ansluten till. Upprepa det här steget om du vill öppna fler anslutningar.

Steg 4: Spara en anslutningsfil för en annan Windows-baserad dator

  1. Markera sessionsfönstret för den andra Windows-baserade dator som du är ansluten till.

  2. Klicka på Inställningar på RDC-menyn.

  3. Välj de alternativ som du vill använda.

  4. Klicka på Spara som på Arkiv-menyn.

  5. Skriv namnet på din anslutningsfil i rutan Spara som.

  6. Klicka på Spara.

  7. Upprepa detta för varje ytterligare Windows-baserad dator som du är ansluten till.

Tips! Klicka på Öppna en sparad anslutning på Arkiv-menyn och klicka sedan på filen du vill använda om du snabbt vill ansluta till en Windows-baserad dator som du sparat en anslutningsfil för.
