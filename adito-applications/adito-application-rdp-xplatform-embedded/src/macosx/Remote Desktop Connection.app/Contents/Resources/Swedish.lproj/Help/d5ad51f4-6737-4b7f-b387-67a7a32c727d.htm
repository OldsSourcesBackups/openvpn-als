Jag hör inte ljuden

Visa alla

Orsak: Ljudinställningarna är inaktiva.

Lösning: Välj ett alternativ för att spela upp ljud.

  1. Klicka på Inställningar på RDC-menyn.

  2. Klicka på Ljud, och klicka sedan på det alternativ du vill använda.

Orsak: Gruppprincipen för Terminal Services tillåter inte att ljud vidarebefordras från den Windows-baserade datorn.

Lösning: Kontakta nätverksadministratören om din grupprincip för Terminal Services.

Grupprincipen åsidosätter alla inställningar som du gör i Anslutning till fjärrskrivbord.Om grupprincipen för Terminal Services inte tillåter vidarebefordring av ljud spelas inte ljuden upp varken på den Windows-baserade datorn eller på Mac-datorn.

Orsak: Högtalarvolymen är låg eller inställd på "ljud av".

Lösning: Kontrollera ljudinställningarna för både Mac-datorn och den Windows-baserade datorn.

Om volymen är avstängd på en av datorerna spelas inte ljudet upp.
